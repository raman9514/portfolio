Project Overview

The haidakhandi.org project is a platform built for a religious community, offering a user-friendly directory for community members. The site allows users to register, provide personal and professional information, and search for other members using various filters. It also offers news updates, event registration, and a global map of the community.

Technologies Used
Backend Development: Python, Django, PostgreSQL
Frontend Development: HTML, Bootstrap5, CSS, JavaScript
Cloud & Deployment: Lightspeed Server
Design & Tools: Figma (for design), Git (version control)
Key Features

User Directory: A comprehensive directory where community members can register and provide detailed personal and professional information.

Search Functionality: Users can search and filter others based on name, profession, or location, allowing for efficient networking.

Global Map Integration: Displays user locations on a map based on their pinned details, showcasing the community's global presence.

Event Registration: Users can register for events directly on the platform, making community gathering organization easier.

News Section: Provides up-to-date news related to the community.

Free Resources: Offers free resources to the community members like ebooks, guides, etc.

Challenges & Solutions

Challenge: Handling large-scale user data and performing complex searches across various attributes like location and profession.

Solution: Integrated PostgreSQL for efficient and rapid search and filtering capabilities, improving performance.

Challenge: Displaying accurate user locations on a map.

Solution: Implemented a mapping system that uses pinned locations to visualize users globally.

Learning Outcome
  • Gained deeper expertise in using PostgreSQL for optimizing search functionalities and handling large datasets.
  • Improved skills in integrating mapping systems and providing user-friendly search and filter features.
