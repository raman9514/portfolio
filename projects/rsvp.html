Project Overview

RSVP by Weddography is a web application designed to simplify event management. Users can create events, add specific functions, manage guest lists, and send out invitations to collect RSVPs and preferences such as food, accommodation, and stay arrangements.

Technologies Used
  • Backend: Django
  • Database: PostgreSQL
  • Frontend: JavaScript, HTML, CSS
Key Features

Event Creation: Users can set up events and add multiple functions for detailed planning.

Guest Management: Allows users to add guest lists, send invitations, and track RSVPs.

Invitation System: Sends invitation links where guests can confirm attendance and provide preferences.

Preference Collection: Collects guest preferences for food, stay, and accommodation.

Challenges & Solutions

Issue: Managing diverse guest preferences for food and accommodation.

Solution: Created an intuitive form interface to gather preferences efficiently while organizing responses for event managers to process easily.

Project Outcome

This project aimed to streamline event planning by reducing the manual effort of tracking guest details and preferences. While the project is complete, it is not live due to client-related issues.
