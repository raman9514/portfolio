Project Overview

The haidakhandi.org platform is designed to support a global religious community, offering a directory of members with personal and professional details. The platform enables users to register, fill out detailed forms, search for other members based on various filters, and view community members' locations on a global map. In addition, it provides news, event registration, and free resources to the community.

Technologies Used
  • Backend: Django, PostgreSQL
  • Frontend: HTML, CSS, JavaScript
  • Cloud Services: AWS (EC2, S3)
  • Deployment Tools: Nginx, Gunicorn
  • Version Control: Git
Key Features

User Directory: Users can register and add personal and professional details, which are visible in the directory.

Advanced Search: Users can search based on criteria such as name, profession, and location.

Global Map Integration: Displays users' pinned locations, showing the global distribution of community members.

Event Registration: Community members can register for upcoming events directly on the platform.

News Section: Provides the latest updates and news related to the community.

Free Resources: Access to free community resources such as e-books and guides.

Challenges & Solutions

Challenge: Efficiently searching and filtering a large database of users by location and profession.

Solution: Integrated PostgreSQL for optimized search functionality, enhancing search performance.

Challenge: Accurate global representation of users based on their pinned locations.

Solution: Implemented a mapping feature that uses user-pinned locations to display on a global map.

Project Outcome

The platform successfully streamlined community management by providing a directory, events registration, and a map of global users. The search functionality, using PostgreSQL, significantly improved user discovery, and the platform facilitated better networking within the community.
