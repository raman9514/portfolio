Customised Protein
Preview Code
Project Overview

CustomisedProtein.com is an e-commerce platform that allows users to personalize their protein supplements based on their fitness goals, dietary restrictions, and taste preferences. The platform offers a seamless product customization experience, from choosing ingredients to configuring protein blends, all the way to a secure and smooth checkout process.

Technologies Used
Backend Development: Python, Django, MySQL
Frontend Development: HTML, Bootstrap5, CSS, JavaScript
Cloud & Deployment: EC2, S3, NGINX, Gunicorn
Design & Tools: Figma (for design), Git (version control)
Key Features

Customizable Protein Blends: Users can create their own protein supplements by selecting ingredients based on their preferences and goals.

Secure Payment Gateway: Integrated with Razorpay API for seamless and secure payment transactions.

Product Recommendations: The platform suggests protein blends based on user data and preferences to simplify decision-making.

Responsive Design: The website is optimized for all devices, ensuring a smooth experience across mobile, tablet, and desktop.

User-Friendly Checkout: Simplified checkout process with multiple payment options and fast order confirmation.

Challenges & Solutions

Challenge 1: Customization Complexity

Allowing users to customize their protein blends while maintaining an intuitive user experience was a complex task.

Solution: We implemented a dynamic form system that guided users through the process step by step, making the customization process simple and enjoyable.

Challenge 2: Secure Payment Handling

Ensuring a secure checkout process for customers while handling payments efficiently.

Solution: Razorpay API was integrated to provide secure payment gateways, ensuring encryption of sensitive data and real-time transaction verification.

Challenge 3: Responsive User Interface

Ensuring the platform functioned smoothly on mobile and desktop devices.

Solution: Bootstrap5 was used to implement a fully responsive layout, ensuring a consistent experience across devices.

Outcome

The Customised Protein platform successfully launched, offering users an easy and intuitive way to create personalized protein blends. The integration of secure payments, responsive design, and efficient product customization led to a positive reception from users. The platform saw high user engagement and repeat customers due to the smooth checkout and personalized product experience.

  • Developed the backend using Python and Django to handle product customization logic, order processing, and user accounts.
  • Integrated third-party APIs such as payment gateways for smooth and secure checkout transactions.
  • Collaborated with designers to create a user-friendly and responsive front-end using Bootstrap5, HTML, and CSS.

Deployment & Hosting
