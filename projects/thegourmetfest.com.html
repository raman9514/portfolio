Project Overview
The Gourmet Fest is a platform designed for catering companies to participate in a nomination-based competition, where they submit their entries, and a jury votes to determine the best in various categories. The system streamlines the nomination and voting process, providing an efficient and seamless experience for both participants and judges.
Technologies Used
Backend Development: Python, Django, MySQL
Frontend Development: HTML, Bootstrap5, CSS, JavaScript
Cloud & Deployment: EC2, S3, NGINX, Gunicorn
Design & Tools: Figma
Key Features

Nomination System: Catering companies can easily submit their nominations through an intuitive form.

Voting Mechanism: A secure and fair voting system allows jury members to cast their votes for nominations.

Admin Dashboard: A backend dashboard for managing nominations, votes, and overall event progress.

Responsive Design: Optimized for different devices, ensuring a smooth user experience across platforms.

Secure Login System: Ensures that only verified users can access the voting and nomination sections.

Challenges & Solutions

Challenge 1: Scalability of the Voting System :Managing a large number of votes in real-time without server overload was crucial.
Solution: We optimized database queries and implemented load balancing on AWS EC2, ensuring smooth processing even under high traffic.

Challenge 2: Secure Data Handling Protecting sensitive user and nomination data from unauthorized access was a priority.
Solution: We implemented SSL encryption and secured all access points using NGINX and authentication systems, ensuring data safety.

Challenge 3: User Experience on Multiple Devices Ensuring that the platform works flawlessly on various devices posed a challenge.
Solution: We used responsive web design techniques with Bootstrap 5, guaranteeing a smooth user experience across mobile, tablet, and desktop devices.
