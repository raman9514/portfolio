Project Overview

The SOS Crowdfunding platform allows users to raise campaigns specifically for educational purposes. It offers a user-friendly interface where donors can easily contribute to campaigns, and students can raise funds for their education. The platform also ensures transparency and security in handling donations, and provides certificates of appreciation to donors.

Technologies Used
Backend Development: Python, Django, MySQL
Frontend Development: HTML, Bootstrap5, CSS, JavaScript
Cloud & Deployment: EC2, S3, NGINX, Gunicorn
Design & Tools: Figma (for design), Git (version control)
Key Features

Campaign Creation: Users can create campaigns for educational purposes and set fundraising goals.

Donor Support: Donors can contribute to campaigns and receive certificates of appreciation via email.

Secure Payments: Integrated Razorpay API for secure and smooth payment processing.

Admin Dashboard: A backend dashboard to monitor campaigns, donations, and overall platform performance.

Responsive Design: Ensured a seamless experience across various devices, including mobile, tablet, and desktop.

Secure Login: Implemented secure login systems to protect user data and ensure platform integrity.

Challenges & Solutions

Challenge 1: Ensuring Secure Payments

Handling payment transactions securely was crucial to build trust with users and donors.

Solution: We integrated Razorpay API, ensuring secure, encrypted transactions with real-time verification.

Challenge 2: Optimizing for High Traffic

Ensuring that the platform could handle a large number of campaigns and donations at peak times was a challenge.

Solution: We used AWS EC2 and optimized the database to ensure the platform could handle high traffic seamlessly.

Challenge 3: User Experience Across Devices

Ensuring that both campaign creators and donors could have a smooth experience on all devices was essential.

Solution: We implemented responsive design techniques using Bootstrap5 to optimize the user interface for different devices.
